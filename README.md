# 📊 Student Social Media & Mental Health Analytics Dashboard

## 📌 Project Overview

This Power BI project analyzes the impact of social media usage on students' mental health, stress levels, sleep patterns, and study habits. The dashboard provides interactive insights to help understand student well-being and support data-driven decision-making.

## 🎯 Project Objectives

* Analyze social media usage patterns among students.
* Evaluate mental health and stress levels.
* Understand the impact of sleep and study habits on well-being.
* Compare student behavior across different countries.
* Generate actionable insights and recommendations.

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

### 5. Country Analytics

* Study Hours by Country

## 🔍 Key Insights

* Higher social media usage is associated with lower mental health scores.
* Better sleep habits contribute to improved mental well-being.
* Stress levels vary across countries and student groups.
* Balanced lifestyle habits positively impact student wellness.

## 💡 Business Recommendations

* Reduce excessive social media usage.
* Improve student sleep habits.
* Provide mental health counseling.
* Encourage healthy lifestyle routines.

## 📊 Project Outcome

The dashboard transforms raw student data into meaningful insights, helping educational institutions improve student wellness, mental health awareness, and academic performance.
